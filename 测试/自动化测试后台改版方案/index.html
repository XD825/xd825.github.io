    自动化测试后台改版方案

    自动化测试后台改版方案

    目的：解决自定义关键字，减少后端的研发压力

    技术选型：

    Django+DRF

    Vue

    MySQL/sqlite

    权限中台：使用公司现有的权限中台

    表结构：

    项目表
    字段 类型 描述
    关键字表
    字段 类型 描述
    模块表
    字段 类型 描述
    用例表
    字段 类型 描述
    套件表
    字段 类型 描述
    夹具表
    字段 类型 描述
    报告表
    字段 类型 描述

    实现接口：

    项目接口：

    GET /projects/ 获取项目列表

    GET /projects/{id}/ 获取单个项目

    POST /projects/ 添加项目

    PUT /projects/{id}/ 修改项目

    PATCH /projects/{id}/ 修改项目

    DELETE /projects/{id}/ 删除项目

    POST /projects/{id}/run

    关键字接口：

    GET /keywords/ 获取关键字列表

    GET /keywords/{id}/ 获取单个关键字信息

    POST /keywords/ 添加关键字

    PUT /keywords/{id}/ 修改关键字

    PATCH /keywords/{id}/ 修改关键字

    模块接口：

    GET /modules/

    GET /modules/{id}/ 获取单个模块

    POST /modules/ 添加模块

    PUT /modules/{id}/ 修改模块

    PATCH /modules/{id}/ 修改模块

    DELETE /modules/{id}/ 删除模块

    POST /modules/{id}/run

    用例接口：

    GET /testcases/

    GET /testcases/{id}/ 获取单个用例

    POST /testcases/ 添加用例

    PUT /testcases/{id}/ 修改用例

    PATCH /testcases/{id}/ 修改用例

    DELETE /testcases/{id}/ 删除用例

    POST /testcases/{id}/run/

    套件接口：

    GET /testsuits/

    GET /testsuits/{id}/ 获取单个套件

    POST /testsuits/ 添加套件

    PUT /testsuits/{id}/ 修改套件

    PATCH /testsuits/{id}/ 修改套件

    DELETE /testsuits/{id}/ 删除套件

    POST /testsuits/{id}/run/

    夹具接口：

    GET /fixtures/ 获取夹具

    GET /fixtures/{id}/ 获取单个夹具

    POST /fixtures/ 添加夹具

    PUT /fixtures/{id}/ 修改夹具

    PATCH /fixtures/{id}/ 修改夹具

    DELETE /fixtures/{id}/ 删除夹具

    报告接口：

    GET /reports/ 获取报告

    GET /reports/{id}/ 获取单个报告

    DELETE /reports/{id}/ 删除报告

    GET /reports/{id}/download/ 下载报告

    简介接口：

    GET /summary/ 获取统计信息
